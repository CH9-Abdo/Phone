## ğŸ› ï¸ How to Manage Products

We have provided a custom Python tool to make managing your inventory easy.

### Using `admin_manager.py`
This is a graphical interface to add, edit, or remove phones from the database (`js/phones_data.js`) without touching code.

1.  **Run the tool:**
    ```bash
    python3 admin_manager.py
    ```
    *(Note: You may need to update this script to point to `js/phones_data.js` instead of `js/script.js` since we split the files).*

### Using `phone_downloader.py`
Need images for new phones? This tool automatically searches GSMArena and downloads high-quality images.

1.  **Run the tool:**
    ```bash
    python3 phone_downloader.py
    ```
2.  Type the phone name (e.g., "Samsung S24").
3.  Click Download.
4.  The images will be saved in the `Photos` folder.
